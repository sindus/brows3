Browse S3 like a
file manager — with thumbnails

A high-performance S3 desktop browser with grid view, progressive image thumbnails, bundled video support, and a local LRU cache. Built with Rust + Tauri.

🍎 macOS (Apple Silicon) 🪟 Windows 🐧 Linux

macOS Intel, .deb and .msi also available on the Releases page.

What's new in this fork

Visual browsing, finally.

Everything that was missing from the original — thumbnails, grid view, and video support — now ships out of the box.

New 🖼️

Grid / Icon View

Toggle between list and grid with a single click. The grid shows thumbnails for images and videos, or file-type icons for everything else.

New ✨

Progressive Thumbnails

Thumbnails load one by one as you browse — iCloud style. No waiting for the whole bucket. Thumbnails appear progressively in real time.

New 🎬

Video Thumbnails

First-frame thumbnails extracted via bundled ffmpeg. No separate install needed — it works on every machine, right out of the box.

New 🔍

Content-Type Detection

Images and videos without file extensions are detected via the S3 Content-Type header. No more missing thumbnails.

New 💾

Local LRU Cache

Thumbnails are cached on disk and reused instantly on subsequent visits. Default limit: 1 GB. Least-recently-used entries are evicted automatically.

New ⚙️

Cache Settings UI

Configure the cache size limit (100 MB – 10 GB) with a slider, and clear the cache with one click — all from the Settings page.

Core features

Everything you expect, and more.

All the power of the original brows3, now with visual browsing on top.

⚡

Instant Navigation

Prefix-indexed caching makes folder traversal instantaneous after the first load.

🔎

Deep Recursive Search

Find any file across millions of objects in milliseconds, even inside nested folders.

📝

Monaco Editor

Edit text, JSON, and code files directly in S3 with VS Code's engine.

📄

Rich Previews

High-res images, videos, PDFs — preview without downloading.

🔒

Secure Profiles

Credentials stored in the OS keychain. Switch AWS accounts or S3-compatible providers instantly.

📦

Bulk Operations

Upload, download, and delete multiple files or entire folder trees at once.

🔗

Presigned URLs

Generate temporary shareable links with configurable expiry directly from the bucket view.

🌐

S3-Compatible Providers

Works with MinIO, Cloudflare R2, Wasabi, DigitalOcean Spaces, and any S3-compatible endpoint.

🔄

Auto-Updates

Signed background updates keep you on the latest secure version automatically.
