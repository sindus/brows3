# Support

Brows3 is a desktop **S3 browser**, **S3 explorer**, and **S3 client** for Amazon S3 and S3-compatible storage.

## Before opening an issue

Please check:

- [README](README.md)
- [Contributing Guide](CONTRIBUTING.md)
- [macOS Troubleshooting](docs/MACOS_TROUBLESHOOTING.md)
- [Release Signing Guide](docs/RELEASE_SIGNING.md)

## Good issue reports include

- Brows3 version
- operating system
- storage provider (AWS S3, MinIO, Cloudflare R2, Wasabi, DigitalOcean Spaces, other)
- exact steps to reproduce
- expected behavior
- actual behavior
- logs or screenshots

## Common support topics

- macOS Gatekeeper says the app is damaged
- profiles or credentials do not persist
- S3-compatible storage endpoint setup
- bucket listing or object browser performance
- updater or release installation problems

## Where to ask

- Use GitHub Issues for bugs, regressions, and concrete feature requests.
- Use existing issues first if your problem matches an open report.
